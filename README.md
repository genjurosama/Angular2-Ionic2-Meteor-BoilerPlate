# Simple Meteor and Ionic application based on Angular 2

Frameworks used:

- Meteor 1.3.2.4
- Ionic 2.0.0-beta.2
- Angular 2.0.0-beta.15

Install dependecies:
```
npm install
```
Run on web browser:
```
npm start
```
Run on browser and android device
```
npm run android-device
```
